Task 3: Create new feature(s)¶

I created 3 new features. Two of the features are scaling email to total emails received/sent while the last feature is around scaling compensation.

In [11]:
for key, value in data_dict.iteritems():
    # fraction from this person to poi
    if value['from_this_person_to_poi'] != "NaN" and value['to_messages'] != "NaN":
        value['fraction_from_this_person_to_poi'] = value['from_this_person_to_poi'] / float(value['to_messages'])
    else:
        value['fraction_from_this_person_to_poi'] = "NaN"
    # fraction from poi to this person
    if value['from_poi_to_this_person'] != "NaN" and value['from_messages'] != "NaN":
        value['fraction_from_poi_to_this_person'] = value['from_poi_to_this_person'] / float(value['from_messages'])
    else:
        value['fraction_from_poi_to_this_person'] = "NaN"
    # fraction total compensation from stocks (assuming you have salary)
    if value['salary'] != "NaN" and value['total_payments'] != "NaN" and value['total_stock_value'] != "NaN":
        value['fraction_compensation_stock'] = value['total_stock_value'] / float(value['total_payments'] + value['total_stock_value'])
    else:
        value['fraction_compensation_stock'] = "NaN"

Task 5: Tune your classifier to achieve better than .3 precision and recall¶

using our testing script. Check the tester.py script in the final project folder for details on the evaluation method, especially the test_classifier function. Because of the small size of the dataset, the script uses stratified shuffle split cross validation. For more info: http://scikit-learn.org/stable/modules/generated/sklearn.cross_validation.StratifiedShuffleSplit.html

using pipeline to explore parameterspace and stratified shuffle split to subsample and validate

from sklearn.pipeline import Pipeline
from sklearn.grid_search import GridSearchCV
from sklearn import tree
from sklearn.cross_validation import StratifiedShuffleSplit
from sklearn.metrics import classification_report

# X_train, X_test, y_train, y_test = sklearn.cross_validation.train_test_split(features, labels, test_size=0.3, random_state=42)

steps = [('kbest', SelectKBest() ),
          ('dtree', tree.DecisionTreeClassifier())]
pipeline = Pipeline(steps)

folds = 100
cv = StratifiedShuffleSplit(labels, n_iter = folds, test_size = 0.4, random_state = 42)

parameters = dict(kbest__k=[2, 3, 4, 5, 6], 
              dtree__criterion = ['gini', 'entropy'],
              dtree__min_samples_split = [2, 3, 4, 5, 10],
              dtree__min_samples_leaf = [1, 2, 3, 4, 5])

clf = GridSearchCV(pipeline, parameters, cv = cv, scoring = 'f1')

clf.fit(features, labels)
y_predictifeaturesons = clf.predict(features)
